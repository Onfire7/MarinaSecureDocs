Map-plotting fully specified. MarinaMap (a marina-wide master map, plus optional per-parent-Location detail maps) and LocationMapPlacement (a join entity, since one Location can be plotted on multiple maps at different coordinates) back this screen's map view and the schematic map mode on Location list / map view — one set of plotted rectangles, two different color overlays (reservation status here, occupancy/type there). Reservation eligibility is unrelated to any of this: it's the existing per-Location reservation_enabled toggle, which is what makes a mixed-use dock work (some Slips reservable short-term, others long-term leased, side by side — see Data Model — Location). The upload + plotting UI lives in Admin's Location Types & Locations setup.
